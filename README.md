# Pdf-Merger

# 📑 PDF Merger Utility

A powerful and simple Python script to combine multiple PDF documents into a single file. Built using the **PyPDF2** library, this tool is perfect for merging reports, certificates, or any scanned documents quickly.

## 📖 Description
This project automates the task of merging PDF files. Instead of using online tools that might compromise your privacy, you can use this local Python script to merge your files securely on your own machine.

## ✨ Key Features
* **Bulk Merging:** Merge two or more PDF files instantly.
* **Order Control:** Merges files in the sequence provided in the code.
* **Secure:** No need to upload sensitive documents to third-party websites.
* **Lightweight:** Uses minimal resources and a single dependency.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Library:** `PyPDF2`
* **Platform:** Jupyter Notebook / Anaconda

## 📁 Project Structure
* `PDF_Merger.ipynb` - The main Jupyter Notebook containing the logic.
* `merged_output.pdf` - The final output file generated after running the script.
* `README.md` - Documentation of the project.
